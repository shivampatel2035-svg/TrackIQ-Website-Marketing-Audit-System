# TrackIQ-Website-Marketing-Audit-System
TrackIQ is a web-based application that analyzes a website's digital marketing setup. It detects marketing tracking tools such as Google Analytics, Google Tag Manager, Meta Pixel, and Microsoft Clarity, performs basic SEO and performance analysis, and generates a marketing health report with recommendations for improvement.

#  TrackIQ

TrackIQ is a full-stack website auditing tool that analyzes websites and generates detailed marketing reports.

It scans websites in real time and provides insights into:

-  Tracking Technologies
-  SEO
-  Performance
-  Downloadable PDF Reports

---

 Features

Website Analysis:-

- Scan any public website
- Real-time website audit
- Dynamic dashboard
- Live analysis results

---

 Tracking Detection

Detects:

- Google Analytics 4 (GA4)
- Google Tag Manager (GTM)
- Meta Pixel
- Microsoft Clarity

---

##  SEO Audit

Checks:

- Title Tag
- Meta Description
- H1 Tag
- Open Graph Tags
- robots.txt
- sitemap.xml

---

## Performance Report

Displays:

- Page Size
- Number of Images
- JavaScript Files
- CSS Files
- Estimated Load Time

---

##  Dashboard

- Overall Website Score
- Tracking Score
- SEO Score
- Performance Score
- Interactive Charts
- Dynamic Recommendations

---

##  PDF Report

Generate and download a professional PDF report containing:

- Website Details
- Audit Results
- Tracking Report
- SEO Report
- Performance Report
- Recommendations

---



#  Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/trackiq.git
```

---

## Backend

```bash
cd trackiq-backend
npm install
npm start
```

Runs on:

```
http://localhost:5000
```

---

## Frontend

```bash
cd trackiqproject-frontend
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```

---

# 📖 Usage

1. Start Backend
2. Start Frontend
3. Open the application
4. Enter any website URL
5. Click **Scan Website**
6. View the generated report
7. Download the PDF report
